# Parfum Site Frontend

The React/Vite client for the Parfum Site full-stack store. It contains the public catalogue, customer authentication screens, localStorage-backed cart and demo checkout, plus the administrator product-management interface.

## Run

Start the API from `../server` first, then:

```bash
npm install
npm run dev
```

Vite proxies `/api` to `http://localhost:5000`.

## Commands

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

See the [root README](../README.md) for complete setup, API, architecture, and security notes.
